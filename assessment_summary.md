## Question Format

Each question follows a consistent format:

```
### Q[N]. [Question text]

**A)** [Option A]
**B)** [Option B]
**C)** [Option C]
**D)** [Option D]

**Answer:** [letter] — [option text]

**Explanation:** [explanation text]

**Topic:** [topic]
```

- Most questions have 4 options (A–D)
- Senior-level questions may include 5 options (A–E) for more nuanced scenarios
- Each question includes a detailed explanation for learning purposes
